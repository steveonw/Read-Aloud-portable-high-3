Keyboard and troubleshooting

F8 reads aloud from the cursor to the end of the draft, highlighting each sentence as it is spoken. The next sentence is prepared while the current one plays, so reading is continuous. With text selected, F8 reads only the selection. Esc stops, and pressing F8 again resumes from the stopped sentence. Editing the text stops playback.

Render / Update synthesizes every sentence of the draft into memory, and Export WAV stitches them into one audio file with natural pauses. After editing, Render / Update re-records only the sentences that changed; everything else is reused instantly. Changing the speed re-renders everything, since it produces different audio.
