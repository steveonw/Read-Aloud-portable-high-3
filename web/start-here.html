Start here

This page identifies the computer and points to the correct offline launcher. Browsers cannot safely start programs automatically, so open the named item from the USB folder.

Windows detected

Double-click START - WINDOWS.exe.

macOS detected

Double-click START - MACOS.app. The first unsigned personal build may need approval in Privacy & Security.

Linux detected

Open START - LINUX.sh. If your file manager opens it as text, run bash "START - LINUX.sh" from a terminal.
